Open Console



How to set up Visual Sudio Code:

1. Click here to go to the VSC download site.
2. Choose the option for your computer type (eg. Mac).





The Console screen is not part of the main webpage. It is used for running tests and printing text which you do not want to be shown on the main screen. You open the console screen by right-clicking the main screen and clicking "inspect". Then, once the inspect screen has appeared on the side, you click on the section "console". To print text on the console screen, you use the function "console.log()".
